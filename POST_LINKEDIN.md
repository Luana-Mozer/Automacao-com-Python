# Postagem para LinkedIn

Finalizei mais um projeto para o meu portfólio: uma automação com Python que transforma uma rotina manual de cadastro de produtos em uma ferramenta com interface gráfica, botões de controle e executável para Windows.

Nesse projeto eu quis ir além do script simples. Eu criei uma solução que:

✅ lê uma base de produtos em CSV  
✅ acessa o navegador automaticamente  
✅ preenche formulários com PyAutoGUI  
✅ permite pausar, continuar e interromper a automação  
✅ possui atalhos de teclado com Ctrl+P e Ctrl+I  
✅ tem interface gráfica com Tkinter  
✅ foi empacotada em um arquivo .exe com PyInstaller  
✅ está documentada com imagens, vídeo e README completo  

O que mais gostei nesse projeto foi perceber como Python pode sair do código e virar uma ferramenta prática, com cara de aplicação mesmo. Para mim, foi uma experiência muito importante porque conectei automação, dados, interface gráfica, organização de projeto e documentação.

Esse tipo de solução mostra exatamente o caminho que eu quero seguir na tecnologia: criar automações que reduzam tarefas repetitivas, organizem processos e ajudem pessoas a ganhar tempo.

Foi um projeto de aprendizado, mas com mentalidade de produto: pensei no uso, no controle da execução, na apresentação visual e na entrega final.

Tecnologias usadas:
Python | Pandas | PyAutoGUI | Tkinter | Keyboard | PyInstaller | CSV | GitHub

Mais um passo na minha evolução como estudante de Engenharia de Software e futura desenvolvedora focada em automação, dados e soluções inteligentes.
